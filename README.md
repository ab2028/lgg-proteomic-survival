# Comparative Survival Modeling of Low-Grade Glioma Patients Using Targeted Proteomic Data


This repository contains the code and analysis for the project: "Comparative Survival Modeling of Low-Grade Glioma Patients Using Targeted Proteomic Data."

- `codes/`    R scripts for data preprocessing, factor analysis, and survival modeling  
- `tables/`     Generated tables used in the manuscript (as .tex files)  
- `figures/`    Generated figures  
- `data/`       Input data
